# BingoJS

SingleJS is an AS2 emulator source written in node java-script and it is open-source as well. The source will be worked on to make it more better, stable and of course exploit free. The main purpose of this source is to help other users in the community to build their own emulators, contributions are accepted.

Current updates:

1. Login server done.
2. Game server done.
3. Added some handlers to get the game running

Required modules:

1. net
2. md5
3. xml2js
4. js-logs
5. mysql2
